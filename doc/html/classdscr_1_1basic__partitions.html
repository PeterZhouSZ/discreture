Detailed Description

template<class IntType>
class dscr::basic_partitions< IntType >

class of partitions of the number n.

Parameters
IntTypeshould be an integral type with enough space to store n and k. It can be signed or unsigned.

Example:

partitions X(6);
for (auto& x : X)
    cout << x << " ";

Prints out: 

[ 1 1 1 1 1 1 ] [ 2 1 1 1 1 ] [ 3 1 1 1 ] [ 2 2 1 1 ] [ 4 1 1 ] [ 3 2 1 ] [ 2 2 2 ] [ 5 1 ] [ 4 2 ] [ 3 3 ] [ 6 ]
