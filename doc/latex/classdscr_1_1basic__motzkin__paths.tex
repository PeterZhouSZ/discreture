Motzkin paths are paths that go from $(0,0)$ to $(0,2n)$, which never go below the $ y=0$ line, in which each step is from $(x,y)$ to either $(x+1,y+1)$ or $(x+1,y-1)$ or $(x+1,y)$ \#\-Example Usage\-: \begin{DoxyVerb}motzkin_paths X(4)
for (const auto& x : X)
    cout << x << endl;
\end{DoxyVerb}
